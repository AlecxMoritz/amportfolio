shadr

shadr was my first fullstack PERN application with an original concept. I used multiple different react packages for my 1.0 version. The backend features encryption, tokenization using JWT, session validation, and over three tables in the database. The client for shadr is a social media styled CRUD application allowing users to sign up, log in, post their 'shades' and 'dislke' other users posts.
